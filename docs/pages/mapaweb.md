# Mapa web interactivo de areas naturales protegidas de Entre Rios
---

**Autores:** Noeli D. Baeza, Jesica E. Rios  
**Institucion:** Centro Regional de Geomatica (CeReGeo), FCyT - UADER  
**Ubicacion geografica:** Provincia de Entre Rios, Argentina  
**Periodo:** 2024

---

## Resumen

Se desarrollo un mapa web interactivo para visualizar el sistema provincial de areas naturales protegidas, incluyendo sitios RAMSAR y parques nacionales.

El proyecto se realizo junto a la Secretaria de Ambiente de Entre Rios, con el objetivo de ampliar el acceso publico a informacion territorial para educacion, gestion y toma de decisiones.

## Metodologia tecnica

- Produccion cartografica en QGIS 3.34.9.
- Publicacion web con complemento QGIS2WEB 3.21.0.
- Implementacion de entorno HTML, CSS y JavaScript con Leaflet.
